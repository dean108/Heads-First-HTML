sans-serif

Font Size

  • Size in pixel from lowest possible point of character to highest
  • Specify base font size as small/medium - IE can scale if it is done in pixels

Font Weight

  • Specify size as normal, bold, etc rather tan 100-900. Numbers aren't supported across all browsers

font-style

  • italic vs oblique - doesn't really matter - browsers will do what they do.

Color

  • only 17 named colors
  • can do RGB(80%, 40%, 0%)
  • or RGB(204, 102,0) (80% of 255, 40% of 255, 0% of 255)
  • 255 is related to values that can be held in one byte
  • hex codes
    • first two digits=red, second two=green , and last two =blue. Hash in front
